The Framework of AVs Safety Assessment Scenario Generation with Traffic Accident-Trigger Vehicle Detection via AI and Real-Driving Kinematic 3D-LiDAR PCD

Minhee Kang, Saeyan Eom(Co-First Author)
Abstraction

In order to introduce new mobility, an assessment that can guarantee SAFETY is needed. Since these assessments are performed based on various scenarios, the scenarios are directly related to safety. Therefore, the scenario needs to be reliable and more specific. This study presents a scenario framework for Automated Vehicle (AVs) safety assessment using Kinematic 3D-LiDAR PCD and AI. Especially, not 2D dimension data or a set of PCD, we focus on individual points of PCD for extracting kinematic features of objects (trigger vehicles). Based on this, vehicles that caused vulnerable situations were classified using AI, and kinematic features such as vehicle location, speed, and TTC, which are essential elements for scenarios, were found to be important. Through this, a logical scenario that secured reliability and concreteness was presented beyond a simple functional scenario.

Classification and Extraction of Potential Accident-causing Vehicles in Accident-vulnerable Situations Based on LiDAR Data in Urban Areas

Minhee Kang, Saeyan Eom(Co-First Author)
The Korea Institue of Intelligent Transport Systems Conference
Abstraction

With the rising adoption of automated vehicles, mixed situations involving manually driven (human-driven) cars are inevitable. Thus, preparations should be made for such diverse situations to render a safe driving environment. This study aims to classify the potentially accident-causing vehicles in traffic using real-world driving data to prepare for dangerous situations. This study focused on physical factors inducing accidents, such as the distance between the ego vehicle and surrounding vehicles and the velocity of surrounding vehicles in actual driving. 3DLiDAR point cloud data from real driving situations were utilized to extract physical features. Moreover, this study proposes deep neural network classification models, which classify potential accident-causing vehicles using the extracted physical features.

Used

Python, open3d, Deep Neural Network

Responsibilities
- DNN classification modelling
- DNN classification models evaluation
